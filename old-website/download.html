Getting The Haskell Cabal

Note: Most people already have Cabal because it is included in the Haskell Platform.

If you want to upgrade to the latest stable version, just run:
$ cabal install cabal cabal-install
Sometimes the older installed version is still on the program search $PATH, you can check you're running the latest version with:
$ cabal --version

May 2016: Release 1.24.0.0

cabal-install tool (version 1.24.0.0)

cabal-install is the command line interface to Cabal and hackage. This is the package that provides the 'cabal' command line program.

Source download: cabal-install-1.24.0.0.tar.gz

Binaries for cabal-install for many platforms are available on Halcyon.

Binary download for Windows (x86): cabal-install-1.24.0.0-i386-unknown-mingw32.zip

Binary download for Windows (x86-64): cabal-install-1.24.0.0-x86_64-unknown-mingw32.zip

Binary download for OS X Yosemite/El Captain (x86_64): cabal-install-1.24.0.0-x86_64-apple-darwin-yosemite.tar.gz

Binary download for Linux (x86): cabal-install-1.24.0.0-i386-unknown-linux.tar.gz

Binary download for Linux (x86-64): cabal-install-1.24.0.0-x86_64-unknown-linux.tar.gz

Binary download for AIX (PowerPC): cabal-1.24.0.0-powerpc-ibm-aix7.1.0.0.xz

The source package requires the Cabal package above and also further packages, which can be found on hackage. To make the process easier the cabal-install tarball contains a bootstrap.sh script which downloads and installs all the dependencies.

Version Numbers

Stable Releases are numbered x.y.z, where y is even. Unstable snapshots between releases are numbered x.y.z, where y is odd. Changes for z are minor.

For example, 1.17.x is the development version leading up to the stable release 1.18.x.
